# Project Detail

## Tech Stack 🛠️

### Core

- ⚛️ React 18 + Next.js 14
- 🎨 Styling: PandaCSS
- 🔄 State Management:
  - Tanstack Query v5 (Server State)
  - Jotai & Zustand (Client State)
- 🌐 Internationalization: next-intl
- 🔒 Authentication: NextAuth.js
- 📚 Storybook 8
- 🧪 TypeScript 5

### Infrastructure

- 🏗️ Monorepo with Turborepo
- 📦 pnpm Workspace (v9)
- 🟢 Node.js (>= 18)

### Apps

- 🌐 Web (@gitanimals/web)
- 👨‍💼 Admin (@gitanimals/admin)
